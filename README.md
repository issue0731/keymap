# Keymap - 단축키 마스터 🎯

지루한 단축키 암기는 이제 그만! Keymap에서 게임처럼 재미있게 엑셀, 포토샵, 윈도우 단축키를 연습하고 당신만의 단축키 지도를 완성하세요.

## 📋 프로젝트 상태

현재 프로젝트는 **기획 단계**입니다.

- ✅ **기획서 완성**: `기획서_Keymap_MVP_v2.0.md` 참고
- 🔄 **개발 대기**: 구현 방식 결정 중

## 🎯 계획된 기능

- **프로그램 선택**: Windows, Excel, Photoshop, PowerPoint
- **퀴즈 모드**: 실시간 키보드 입력 감지
- **목록 모드**: 검색 가능한 단축키 목록
- **라이트/다크 모드**: 테마 전환
- **반응형 디자인**: 모바일부터 데스크톱까지
- **수익화**: Google AdSense, 카카오 애드핏
